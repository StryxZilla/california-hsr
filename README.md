## The Story in Numbers

| Metric | 2008 Promise | 2026 Reality |
|--------|--------------|--------------|
| Cost | $33 billion | $89-128 billion |
| Timeline | Complete by 2020 | Maybe IOS by 2033 |
| Passengers | Full service | Zero |
| Track | 800+ miles | 70 miles complete |

## What This Site Covers

### 1. Money Spent
- Total budget evolution from $33B to $128B
- Actual spending to date ($13.8B)
- Cost per mile analysis
- Comparison to other mega-projects

### 2. Timeline Analysis
- Original vs actual progress
- How projected completion dates have slipped
- Key milestones and delays

### 3. Construction Progress
- Miles of track laid/under construction
- Interactive map of route segments
- Structure completion status

### 4. Who Got the Money
- Top contractors and contract values
- Contract growth (original bid vs current)
- Notable cost overruns

## Sources

All data sourced from:
- California High-Speed Rail Authority official reports
- Business Plans (2008-2024)
- Wikipedia (with primary source citations)
- LA Times investigations
- Federal government announcements

Every data point includes a clickable source link for verification.

## Technical Stack

- Pure HTML/CSS/JavaScript
- D3.js v7 for visualizations
- 538/Vox-inspired design (Roboto Mono, clean grids, annotations)
- Fully responsive
- No build tools required

## Style Guide

Following the example at https://stryxzilla.github.io/maternal-mortality/us-deep-dive.html:
- Clean narrative flow
- Annotations explaining key insights
- Interactive elements with tooltips
- Every chart tells part of the story
